Student Pass/Fail Prediction
Project Name
Student Pass/Fail Prediction using IBM SPSS Modeler
1. Objective
The objective of this project is to predict whether a student will **pass or fail** based on various input factors such as:
- Attendance
- Study hours
- Internal marks
- Previous performance
This helps in:
- Identifying weak students early
- Improving academic performance
- Supporting decision-making for teachers

 2.Tools & Technologies Used
- IBM SPSS Modeler
- Dataset (Student academic data)
- Machine Learning Techniques

 3.Steps Involved
  1. Data Collection
  Collected student data including:
- Marks
- Attendance
- Study hours
- Other academic details
  2. Data Preparation
- Cleaned missing values
- Selected relevant features
- Converted data into proper format
  3. Model Building
- Used SPSS Modeler to create prediction model
- Applied classification techniques such as:
  - Decision Tree
  - Logistic Regression (if used)
  4. Model Training
- Trained the model using dataset
- Adjusted parameters for better accuracy
  5. Model Evaluation
- Checked accuracy of prediction
- Compared predicted vs actual results
  6. Prediction
- Model predicts whether a student will:
  -  Pass
  -  Fail

4. Output
The model provides:
- Pass/Fail prediction
- Accuracy results
- Performance insights

# Student Pass/Fail Prediction Using IBM SPSS Modeler

## Overview

This project predicts whether a student will pass or fail using machine learning techniques in IBM SPSS Modeler. The prediction is based on various academic factors such as attendance, study hours, internal marks, and previous academic performance.

The system helps educational institutions identify weak students early and improve academic decision-making using predictive analytics.

---

# Objective

The objective of this project is to predict student performance based on important academic parameters such as:

* Attendance
* Study Hours
* Internal Marks
* Previous Performance

The project helps in:

* Identifying weak students early
* Improving academic performance
* Supporting teachers in academic decision-making
